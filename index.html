Hey, I’m Ajay Dhangar !

A Web Developer & Web Designer

Ajay Dhangar is a B.Tech CSE student with a passion for web development. With a hobby and work experience in this field, Ajay's dream is to become a Software Development Engineer (SDE). He is proficient in Data Structures and Algorithms with Java and has excellent problem-solving skills using this programming language.

As a front-end developer, Ajay is familiar with HTML, CSS, JavaScript, and React, and is constantly keeping up with the latest technologies and trends in this area. For back-end development, he relies on MongoDB, NodeJs, and Express.js. His extensive knowledge in these areas allows him to create high-quality and efficient web applications.

When not working on web development projects, Ajay enjoys learning new skills and expanding his knowledge in the tech industry. He is an ambitious individual who is dedicated to achieving his goals and making a positive impact in his field.
